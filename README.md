# Lend on Stellar – Architecture Documentation

This repository documents the architecture of the Lend protocol deployed on the Stellar network.

Lend is a real estate tokenization platform enabling compliant issuance of tokenized bonds backed by real-world assets. The Stellar integration relies on Soroban smart contracts, an event-indexing worker infrastructure and a frontend allowing investors to participate in tokenized operations.

This repository provides an overview of the architecture and system design used to deploy Lend on Stellar.
